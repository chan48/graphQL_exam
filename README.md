# GraphQL 예제
유데미 graphql-with-react-course 따라하기.


##실행방법
사전에 node module install 필수.
```
npm install
``` 


###json-server 실행
```
npm run json:server
```


### 서버 실행
```
npm run dev
```



### url 및 파일 정보
- json-server
    - 테스트 URL : http://localhost:3000/users/41
    - data source : ./db.json
- graphiQL
    - URL : http://localhost:4000/graphql
    - query sample : ./sample-query.md
